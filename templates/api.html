Credit Intelligence API

Comprehensive documentation for Quorel's credit scoring and financial intelligence API endpoints.

API Endpoints

Base Information Health Check Score Page API - Score (JSON) Peer Comparison Timeline Events Page API - Events (JSON) Score History Score Trend Glossary

Base Information

BASE http://<host>:5000

This API provides access to credit scoring, peer comparison, event sentiment, and score trends for financial tickers.

Health Check

GET /health

Quick status check to ensure the server is running.

Response

{
  "status": "ok"
}

Score Page

GET /score

Computes creditworthiness score, renders charts, and explanations.

Parameters

Name Type Description
ticker string Stock symbol (default: AAPL)

Response

HTML page with:

  • Final Score (0-100)
  • Feature Importances Chart
  • Price + Moving Averages
  • Sentiment Chart
  • Basic + Advanced Explanations

API - Score (JSON)

GET /api/score/<ticker>

Example Response

{
  "ticker": "AAPL",
  "score_0_to_100": 72.5,
  "p_good_last_step": 0.823,
  "feature_importances": {
    "Return": 0.45,
    "Volatility": 0.32,
    "Liquidity": 0.23
  },
  "explanation": "Model found short-term momentum strong and volume stable."
}

Peer Comparison

GET /peer

Parameters

Name Type Description
symbol string Stock symbol (default: AAPL)

Compares the given ticker against its peers and plots scores.

Timeline (Score Evolution)

GET /timeline

Plots a timeline of scores and logged events.

Response: HTML page with matplotlib chart.

Events Page

GET /events

Shows recent market events and their sentiment.

API - Events (JSON)

GET /api/events

Example Response

{
  "events": [
    {
      "headline": "Apple launches new iPhone",
      "sentiment_label": "pos",
      "sentiment_score": 0.78
    },
    {
      "headline": "Supply chain disruptions hit tech sector",
      "sentiment_label": "neg",
      "sentiment_score": -0.65
    }
  ]
}

Score History (SQLite)

GET /api/score_history/<ticker>

Example Response

[
  {"time": "2025-08-18 15:23:10", "score": 72.5},
  {"time": "2025-08-18 16:02:55", "score": 70.8},
  {"time": "2025-08-19 09:15:01", "score": 74.2}
]

Score Trend Page (Interactive)

GET /score_trend/<ticker>

Renders an interactive chart (Chart.js) of historical score trends.

Glossary

GET /glossary

Displays glossary of financial and credit terms.
